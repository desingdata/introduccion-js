🔁 Iteraciones en JavaScript

Proyecto práctico comentado — definición, código y demo en vivo

01 — Bucle for

Repite un bloque un número determinado de veces. Tiene tres partes: inicialización, condición y actualización.

// for (inicialización; condición; actualización)
for (let i = 1; i <= 5; i++) {
  // Se ejecuta mientras i <= 5
  console.log(`Iteración número: ${i}`);
}

02 — Bucle while

Ejecuta un bloque mientras una condición sea verdadera. La condición se evalúa antes de cada iteración.

let contador = 0;

while (contador < 4) {
  // Se repite mientras contador < 4
  console.log(`Contador: ${contador}`);
  contador++; // evita un bucle infinito
}
— presiona ejecutar —

03 — Bucle do...while

La condición se evalúa después de ejecutar el bloque. Siempre corre al menos una vez.

let n = 0;

do {
  // Ejecuta primero, luego verifica la condición
  console.log(`Valor: ${n}`);
  n++;
} while (n < 3);

04 — Bucle for...of

Recorre los valores de iterables (arrays, strings, Sets). Más legible que el for clásico para colecciones.

const frutas = ["🍎 Manzana", "🍌 Banano", "🍇 Uva"];

for (const fruta of frutas) {
  // fruta toma cada valor del array
  console.log(`Fruta: ${fruta}`);
}

05 — Bucle for...in

Recorre las claves (propiedades) de un objeto. Ideal para objetos literales.

const persona = {
  nombre: "Carlos",
  edad: 28,
  ciudad: "Pereira"
};

for (const clave in persona) {
  // clave toma el nombre de cada propiedad
  console.log(`${clave}: ${persona[clave]}`);
}

06 — Método forEach()

Ejecuta una función por cada elemento del array. No retorna un nuevo array; se usa para efectos secundarios.

const numeros = [10, 20, 30, 40];

// Recibe: valor, índice, array completo
numeros.forEach((num, indice) => {
  console.log(`[${indice}] → ${num}`);
});
— presiona ejecutar —

07 — Método map()

Transforma cada elemento y retorna un nuevo array con los resultados. No modifica el original.

const precios = [100, 200, 350];

// Aplica impuesto del 19% a cada precio
const conImpuesto = precios.map(p => +(p * 1.19).toFixed(2));

console.log("Originales:", precios);
console.log("Con impuesto:", conImpuesto);

08 — Método filter()

Crea un nuevo array con solo los elementos que pasen la condición dada (retornen true).

const edades = [15, 22, 17, 30, 14, 25];

// Solo conserva mayores de 18
const mayores = edades.filter(edad => edad >= 18);

console.log("Todas:", edades);
console.log("Mayores de edad:", mayores);

09 — Método reduce()

Acumula todos los valores en un único resultado. Recibe acumulador, elemento actual y valor inicial.

const ventas = [500, 300, 850, 120];

// acum: acumulador, actual: valor actual, 0: valor inicial
const total = ventas.reduce((acum, actual) => acum + actual, 0);

console.log("Ventas:", ventas);
console.log("Total acumulado: $" + total);

10 — Control de flujo: break y continue

break detiene el bucle completamente. continue salta la iteración actual y pasa a la siguiente.

for (let i = 1; i <= 10; i++) {

  if (i === 4) {
    console.log("⏭ Saltando el 4 (continue)");
    continue; // omite esta iteración
  }

  if (i === 7) {
    console.log("🛑 Deteniendo en 7 (break)");
    break; // termina el bucle
  }

  console.log(`i = ${i}`);
}
