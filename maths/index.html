01

Aritmética básica

Código comentado
Los operadores aritméticos realizan las operaciones matemáticas fundamentales: suma, resta, multiplicación, división y módulo.
// Operadores aritméticos básicos
const a = 10, b = 3;

// Suma: añade dos valores
console.log(a + b);   // → 13

// Resta: diferencia entre valores
console.log(a - b);   // → 7

// Multiplicación
console.log(a * b);   // → 30

// División: cociente exacto
console.log(a / b);   // → 3.333...

// Módulo: resto de la división
console.log(a % b);   // → 1

// Potencia: base elevada al exp.
console.log(a ** b);  // → 1000

02

Objeto Math — Álgebra

Código comentado
El objeto global Math provee constantes y funciones matemáticas sin necesidad de instanciarse.
// Math.abs → valor absoluto
Math.abs(-7);       // 7

// Math.sqrt → raíz cuadrada
Math.sqrt(144);     // 12

// Math.pow → potenciación
Math.pow(2, 8);     // 256

// Math.log → logaritmo natural
Math.log(Math.E);  // 1

// Math.PI → constante π
Math.PI;            // 3.14159...

// Math.random → número [0,1)
Math.random();

// Math.floor/ceil/round → redondeo
Math.floor(4.9);   // 4
Math.ceil(4.1);    // 5

03

Trigonometría

Código comentado
Las funciones trigonométricas de JS trabajan en radianes. Para convertir desde grados: rad = deg × (π/180).
// Conversión de grados a radianes
const toRad = deg => deg * (Math.PI / 180);

// Seno: relación opuesto/hipotenusa
Math.sin(toRad(90));  // 1

// Coseno: relación adyacente/hipotenusa
Math.cos(toRad(0));   // 1

// Tangente: opuesto/adyacente
Math.tan(toRad(45));  // ≈ 1

// Hipotenusa con Pitágoras
const hip = (a, b) =>
  Math.sqrt(a**2 + b**2);
hip(3, 4);  // 5

Estadística descriptiva

Código comentado
Las medidas de tendencia central resumen un conjunto de datos en un solo valor representativo.
const datos = [4,8,6,5,3,2,8,9,10,4];

// Media: suma dividida entre total
const media = arr =>
  arr.reduce((s,n) => s+n, 0) / arr.length;

// Mediana: valor central ordenado
const mediana = arr => {
  const s = [...arr].sort((a,b)=>a-b);
  const m = Math.floor(s.length/2);
  return s.length%2 ? s[m] : (s[m-1]+s[m])/2;
};

// Varianza: dispersión respecto a la media
const varianza = arr => {
  const m = media(arr);
  return media(arr.map(x => (x-m)**2));
};

// Desviación estándar: √varianza
const desv = arr => Math.sqrt(varianza(arr));

Sistema binario y operadores bit a bit

Código comentado
JavaScript permite manipular los bits de los números directamente con operadores bitwise, usados en flags, máscaras y gráficos.
const n = 42;

// toString(2) convierte a binario
n.toString(2);   // "101010"

// AND: 1 solo si ambos bits son 1
12 & 10;        // 8  (1100 & 1010)

// OR: 1 si al menos un bit es 1
12 | 10;        // 14 (1100 | 1010)

// XOR: 1 solo si los bits difieren
12 ^ 10;        // 6  (1100 ^ 1010)

// NOT: invierte todos los bits
~5;             // -6

// Desplazamiento izquierda (×2)
1 << 4;         // 16

// Desplazamiento derecha (÷2)
16 >> 2;        // 4

Gráficas con Canvas

Código comentado
Usando el API Canvas de HTML5 podemos graficar funciones matemáticas al convertir coordenadas matemáticas en píxeles.
// Graficar f(x) = sin(x) en canvas
function graficar(canvas, fn) {
  const ctx = canvas.getContext('2d');
  const {width: W, height: H} = canvas;

  // Limpiar lienzo
  ctx.clearRect(0, 0, W, H);

  // Centro del canvas como origen
  const cx = W/2, cy = H/2;

  ctx.beginPath();
  for (let px = 0; px < W; px++) {
    // Convertir px a radianes
    const x = (px - cx) / 40;
    // Evaluar función y escalar
    const y = cy - fn(x) * 40;
    px === 0 ? ctx.moveTo(px,y)
              : ctx.lineTo(px,y);
  }
  ctx.stroke();
}
