🔀 Condicionales en JavaScript

Proyecto práctico — código comentado + demos interactivos

Concepto

Declaraciones Condicionales

Condicional: Estructura que permite ejecutar bloques de código solo si se cumple una condición. Son la base de la lógica de decisiones en cualquier programa. 
// Las condicionales evalúan expresiones booleanas
// (true / false) para decidir qué código ejecutar.

let llueve = true;   // ← condición

if (llueve) {
  console.log("☂️ Lleva paraguas");
} else {
  console.log("☀️ Día despejado");
}
// Resultado: "☂️ Lleva paraguas"
Cuándo

¿Cuándo usar condicionales?

Se usan cuando el programa debe tomar decisiones según el estado de variables, entrada del usuario, resultados de operaciones u otras condiciones en tiempo de ejecución. 
// ✅ Validar formularios
let edad = 17;
if (edad >= 18) { /* acceso permitido */ }

// ✅ Controlar el flujo de la app
let logueado = false;
if (logueado) { /* mostrar dashboard */ }

// ✅ Comparar resultados
let puntaje = 85;
if (puntaje >= 60) { /* aprobado */ }

// ✅ Manejar distintos roles
let rol = "admin";
if (rol === "admin") { /* acceso total */ }
if

La instrucción if

if: Ejecuta un bloque de código únicamente cuando la condición entre paréntesis evalúa a true. Si es false, el bloque se omite. 
// Sintaxis:  if (condición) { bloque }

let temperatura = 35; // °C

if (temperatura > 30) {
  // Se ejecuta SOLO cuando temp > 30
  console.log("🌡️ Hace calor");
}

// Condición compuesta con && (Y lógico)
if (temperatura > 30 && temperatura < 40) {
  console.log("Calor moderado");
}
else

La instrucción else

else: Define un bloque alternativo que se ejecuta cuando la condición del if es false. Garantiza que siempre haya una respuesta. 
// Sintaxis: if (cond) { ... } else { ... }

let nota = 55;

if (nota >= 60) {
  // Se ejecuta cuando nota ≥ 60
  console.log("✅ Aprobado");
} else {
  // Se ejecuta cuando nota < 60
  console.log("❌ Reprobado");
}
// Resultado: "❌ Reprobado"
else if

La declaración else if

else if: Permite encadenar múltiples condiciones. Se evalúan en orden; cuando una es true, las siguientes se omiten completamente. 
// Convierte nota numérica a calificación letra
let n = 78;

if (n >= 90) {
  console.log("A — Excelente");
} else if (n >= 80) {
  console.log("B — Muy bueno");
} else if (n >= 70) {
  console.log("C — Bueno"); // ← se ejecuta
} else if (n >= 60) {
  console.log("D — Suficiente");
} else {
  console.log("F — Reprobado");
}
switch

La instrucción switch

switch: Evalúa una expresión y la compara con múltiples case usando ===. break impide la ejecución en cascada hacia los casos siguientes. default se ejecuta si ningún case coincide. 
let dia = 3; // 1=Lun … 7=Dom

switch (dia) {
  case 1:
    console.log("Lunes"); break;
  case 2:
    console.log("Martes"); break;
  case 3:
    console.log("Miércoles"); break; // ←
  case 6:
  case 7:
    // Cases agrupados = fin de semana
    console.log("🎉 Fin de semana"); break;
  default:
    console.log("Día inválido");
}
? :

Operador Ternario  condición ? valorTrue : valorFalse

Operador ternario: Versión compacta de if/else en una sola línea. Si la condición es true devuelve el primer valor; si es false, el segundo. Ideal para asignaciones simples; evitar para lógica compleja. 
// Sintaxis: condición ? valorSiTrue : valorSiFalse

let edad = 20;

// Equivale a un if/else en una sola línea:
let acceso = edad >= 18
  ? "✅ Mayor de edad"
  : "🚫 Menor de edad";

console.log(acceso); // "✅ Mayor de edad"

// Ternario anidado (con moderación)
let nota = 75;
let letra = nota >= 90 ? "A"
           : nota >= 80 ? "B"
           : nota >= 70 ? "C"    // ← se ejecuta
           : nota >= 60 ? "D"
           :              "F";

// Ternario en clases CSS dinámicas
let activo = true;
let clase = activo ? "btn-primary" : "btn-disabled";
