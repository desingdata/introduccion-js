01
Variables
var

Una variable es un contenedor con nombre que almacena un valor en memoria. Su contenido puede cambiar durante la ejecución del programa. En JS, toda variable apunta a un espacio de memoria donde se guarda el dato asignado.

variables.js 
// Variable: espacio en memoria con nombre y valor
var ciudad = "Bogotá";
var poblacion = 8000000;
var esCapital = true;

// var tiene alcance de función (function scope)
// y puede redeclararse en el mismo ámbito
var ciudad = "Medellín";  // ✓ permitido con var

console.log(ciudad);      // → "Medellín"
console.log(poblacion);   // → 8000000
console.log(esCapital);   // → true
ciudad → "Medellín"
poblacion → 8000000
esCapital → true
02
Identificadores
naming

Un identificador es el nombre que se le asigna a una variable, función o clase. Debe comenzar con letra, _ o $; no puede empezar con número ni contener espacios. JavaScript distingue mayúsculas y minúsculas (case-sensitive).

identificadores.js 
// ✅ Identificadores VÁLIDOS
var nombre         = "Ana";
var _privado       = 42;
var $precio        = 19.99;
var nombreCompleto = "Ana López";  // camelCase ← convención JS
var edad2024       = 30;          // número en medio: ✓

// ❌ Identificadores INVÁLIDOS (causarían SyntaxError)
// var 2nombre = "error";   ← empieza con número
// var mi nombre = "error"; ← contiene espacio
// var for = "error";       ← palabra reservada

// ⚠️  Case-sensitive: estas son 3 variables DISTINTAS
var color  = "rojo";
var Color  = "azul";
var COLOR  = "verde";

console.log(color, Color, COLOR);  // → rojo azul verde
03
Declaración de Variables
declarar

Declarar una variable es anunciarle al motor de JS que existirá un espacio de memoria con ese nombre. Se puede declarar sin valor inicial (el valor será undefined), y luego asignar o reasignar el valor en cualquier momento.

declaracion.js 
// 1. Solo declaración (sin valor) → undefined
let puntaje;
console.log(puntaje);          // → undefined

// 2. Declaración + inicialización en una línea
let nivel = 1;
console.log(nivel);             // → 1

// 3. Asignación posterior
puntaje = 100;
console.log(puntaje);          // → 100

// 4. Reasignación de valor
nivel = 2;
console.log(nivel);             // → 2

// 5. Declaración múltiple en una sola línea
let x = 10, y = 20, z = 30;
console.log(x + y + z);        // → 60
puntaje inicial → undefined
puntaje final → 100
x+y+z → 60
04
Let
let

let es la forma moderna y recomendada de declarar variables mutables. A diferencia de var, tiene block scope (alcance de bloque { }), no puede redeclararse en el mismo ámbito y no está disponible antes de su declaración (Temporal Dead Zone).

let.js 
// let → block scope: solo existe dentro de { }
let temperatura = 25;

if (true) {
  let temperatura = 30;          // Variable diferente (bloque propio)
  console.log(temperatura);       // → 30
}
console.log(temperatura);         // → 25 (la de afuera no cambia)

// let en bucles: cada iteración tiene su propio scope
for (let i = 0; i < 3; i++) {
  console.log("iteración:", i);  // → 0, 1, 2
}
// console.log(i); → ReferenceError: i no existe aquí

// ✅ Reasignación permitida
let contador = 0;
contador++;
console.log(contador);            // → 1

// ❌ Redeclaración NO permitida
// let contador = 5; → SyntaxError
05
Constantes
const

Una constante (const) es una variable cuya referencia no puede reasignarse después de su declaración inicial. Tiene block scope igual que let. Es obligatorio inicializarla al declararla. Si almacena un objeto o array, su contenido interno sí puede modificarse.

constantes.js 
// const → valor inmutable (no reasignable)
const PI          = 3.14159;
const PAIS        = "Colombia";
const MAX_INTENTOS = 3;

console.log(PI);           // → 3.14159

// ❌ Reasignación NO permitida
// PI = 3; → TypeError: Assignment to constant variable

// ⚠️  Con objetos: la referencia es fija, pero el contenido NO
const usuario = {
  nombre: "Carlos",
  edad: 28
};

usuario.edad = 29;         // ✅ Se puede modificar propiedad
console.log(usuario.edad);  // → 29

// ⚠️  Con arrays: igual, se puede modificar el contenido
const colores = ["rojo", "azul"];
colores.push("verde");     // ✅ permitido
console.log(colores);       // → ["rojo", "azul", "verde"]
