Introducción a JavaScript

Bucles en JavaScript

Aprende las estructuras de iteración más importantes del lenguaje con ejemplos ejecutables y comentarios explicativos.

for

Bucle for

Repite un bloque un número determinado de veces. Ideal cuando conocemos la cantidad de iteraciones de antemano.

// BUCLE FOR
// Sintaxis: for(inicio; condición; actualización)
// - inicio: se ejecuta una vez antes del bucle
// - condición: se evalúa antes de cada iteración; si es false, el bucle termina
// - actualización: se ejecuta al final de cada iteración

for (let i = 1; i <= 5; i++) {
  // El cuerpo del bucle se ejecuta mientras i <= 5
  console.log(`Iteración número: ${i}`);
}

// Recorrer un array con for clásico
// arr.length devuelve la cantidad de elementos del array
const frutas = ['🍎 Manzana', '🍌 Banana', '🍊 Naranja'];

for (let i = 0; i < frutas.length; i++) {
  console.log(`Fruta [${i}]: ${frutas[i]}`);
}
// output
while

Bucle while

Repite un bloque mientras una condición sea verdadera. La condición se evalúa ANTES de cada iteración.

// BUCLE WHILE
// Se usa cuando no sabemos cuántas iteraciones serán necesarias.
// ¡Atención! Si la condición nunca se vuelve false → bucle infinito.

let contador = 1;
let suma = 0;

// Mientras contador sea menor o igual a 10, seguimos sumando
while (contador <= 10) {
  suma += contador; // suma = suma + contador
  contador++;       // incrementamos para evitar bucle infinito
}

// Resultado: suma de 1 a 10 = 55
console.log(`Suma del 1 al 10 = ${suma}`);
// output
do

Bucle do...while

Similar al while, pero la condición se evalúa DESPUÉS de ejecutar el bloque. Garantiza al menos una ejecución.

// BUCLE DO...WHILE
// El bloque se ejecuta al menos UNA VEZ, sin importar la condición.
// Útil para pedir datos al usuario hasta que sean válidos.

let intentos = 0;
let numero;

do {
  intentos++;
  // Simulamos un número aleatorio entre 1 y 10
  numero = Math.floor(Math.random() * 10) + 1;
  console.log(`Intento ${intentos}: número = ${numero}`);
} while (numero !== 7); // repite hasta obtener el 7

console.log(`¡Encontré el 7 en ${intentos} intento(s)!`);
// output
in

Bucle for...in

Itera sobre las propiedades enumerables de un objeto. Devuelve las claves (keys), no los valores.

// BUCLE FOR...IN
// Diseñado para objetos (no arrays). Itera sobre las PROPIEDADES (keys).
// Para cada propiedad, podemos acceder a su valor con objeto[key].

const estudiante = {
  nombre: 'Ana García',
  edad: 22,
  carrera: 'Ingeniería',
  promedio: 9.5
};

// 'propiedad' toma el nombre de cada key en cada iteración
for (let propiedad in estudiante) {
  console.log(`${propiedad}: ${estudiante[propiedad]}`);
  // estudiante[propiedad] → acceso dinámico con notación de corchetes
}
// output
of

Bucle for...of

Itera sobre valores de objetos iterables: arrays, strings, Maps, Sets. La forma más moderna y limpia de recorrer colecciones.

// BUCLE FOR...OF (ES6+)
// A diferencia de for...in, devuelve directamente los VALORES.
// Funciona con cualquier objeto iterable.

const lenguajes = ['JavaScript', 'Python', 'Rust', 'Go'];

// 'lenguaje' recibe cada VALOR del array directamente
for (const lenguaje of lenguajes) {
  console.log(`→ ${lenguaje}`);
}

// También funciona con strings: itera caracter por caracter
const texto = 'LOOP';
for (const letra of texto) {
  console.log(`Letra: ${letra}`);
}
// output
break

Sentencia break

Interrumpe y sale completamente del bucle en el que se encuentra, sin importar si la condición aún es verdadera.

// SENTENCIA BREAK
// Detiene la ejecución del bucle inmediatamente.
// Muy útil en búsquedas: cuando encontramos lo que buscamos, salimos.

const numeros = [3, 7, 12, 5, 8, 20];
const buscar = 12;

for (let i = 0; i < numeros.length; i++) {
  console.log(`Revisando: ${numeros[i]}`);

  if (numeros[i] === buscar) {
    // Encontramos el número → salimos del bucle inmediatamente
    console.log(`✓ Encontrado ${buscar} en índice [${i}]`);
    break; // el bucle termina aquí
  }
}

console.log('Búsqueda finalizada.');
// output
cont

Sentencia continue

Salta la iteración actual y pasa a la siguiente, sin salir del bucle. Útil para filtrar o ignorar ciertos valores.

// SENTENCIA CONTINUE
// No sale del bucle, solo "salta" la iteración actual.
// El bucle continúa con el siguiente ciclo normalmente.

console.log('Números impares del 1 al 10:');

for (let i = 1; i <= 10; i++) {
  if (i % 2 === 0) {
    // Si el número es par, saltamos esta iteración
    continue; // vuelve al inicio del for con i++
  }
  // Solo llegamos aquí si el número es IMPAR
  console.log(`  → ${i}`);
}
// output
