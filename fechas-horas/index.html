1️⃣ Crear un objeto Date

new Date() crea un objeto que representa un momento en el tiempo. Sin argumentos captura ahora; con argumentos define una fecha específica.

// Sin args → fecha/hora actual
const ahora = new Date();

// Con string ISO
const navidad = new Date("2025-12-25");

// Con (año, mes-0, día, h, m, s)
const d = new Date(2000, 0, 1);

2️⃣ Timestamp (Unix Time)

Número de milisegundos desde el 1 ene 1970 (Epoch). Es la representación más universal e interoperable de un instante en el tiempo.

// Date.now() → timestamp actual
const ts = Date.now();

// .getTime() → timestamp del Date
const t = new Date().getTime();

// Timestamp 0 → Epoch
new Date(0); // 1 ene 1970 UTC
▶ Resultado:Cargando...

3️⃣ Modificar fechas (set*)

Los métodos set* mutan el objeto Date, cambiando componentes individuales sin necesidad de crear un nuevo objeto.

const d = new Date();
// Cambiar el año
d.setFullYear(2030);
// Cambiar mes (0=Enero)
d.setMonth(0);
// Cambiar día del mes
d.setDate(1);
// Cambiar hora, min, seg
d.setHours(12, 0, 0);

4️⃣ Formatear fechas

Métodos built-in para convertir un Date a texto. toLocaleDateString() adapta el formato al idioma y región indicada automáticamente.

const d = new Date();
// Fecha según idioma
d.toLocaleDateString("es-MX");
// Solo la hora
d.toLocaleTimeString("es-MX");
// Estándar ISO 8601
d.toISOString();
// Texto completo legible
d.toString();

5️⃣ Intl.DateTimeFormat

API para formateo avanzado e internacionalizado de fechas con control total sobre componentes, idioma y zona horaria.

6️⃣ Diferencia entre fechas

Restar dos objetos Date devuelve milisegundos. Dividiendo entre las constantes adecuadas se obtienen días, horas o minutos.

const inicio = new Date("2025-01-01");
const hoy = new Date();
// Diferencia en milisegundos
const ms = hoy - inicio;
// → días (÷ 86_400_000)
const dias = Math.floor(ms / 86400000);
// → horas (÷ 3_600_000)
const hrs = Math.floor(ms / 3600000);

7️⃣ Countdown (Cuenta regresiva)

Combinar diferencia de fechas con setInterval() crea un temporizador que se actualiza cada segundo mostrando el tiempo restante.

function countdown(target) {
  const diff = new Date(target) - Date.now();
  if (diff <= 0) return "¡Llegó!";
  const d = Math.floor(diff/86400000);
  const h = Math.floor(diff%86400000/3600000);
  const m = Math.floor(diff%3600000/60000);
  const s = Math.floor(diff%60000/1000);
  return `${d}d ${h}h ${m}m ${s}s`;
}

8️⃣ Date.parse() y Validación

Date.parse() convierte un string a timestamp. Retorna NaN si el string no es una fecha válida, permitiendo validar entradas del usuario.

// String → timestamp en ms
Date.parse("2025-06-15"); // número
Date.parse("texto");       // NaN

// Función de validación
function esValida(str) {
  return !isNaN(Date.parse(str));
}
esValida("2025-06-15"); // true
esValida("2025-13-01"); // false

9️⃣ Zonas Horarias (UTC)

getTimezoneOffset() devuelve los minutos de diferencia entre la hora local y UTC. toLocaleString() acepta una zona horaria para mostrar la hora de cualquier región.

const d = new Date();
// Diferencia local vs UTC (minutos)
d.getTimezoneOffset(); // ej: 300 = UTC-5
// Fecha en UTC universal
d.toUTCString();
// Mostrar en zona específica
d.toLocaleString("es-MX", {
  timeZone: "America/Mexico_City"
});
d.toLocaleString("es-ES", {
  timeZone: "Europe/Madrid"
});

🔟 Comparar fechas correctamente

Dos objetos Date nunca son iguales con === aunque representen el mismo instante. La comparación correcta es mediante sus timestamps o los operadores < >.

const a = new Date("2025-06-01");
const b = new Date("2025-12-31");

// ❌ MAL: compara referencias
a === b; // siempre false

// ✅ BIEN: comparar timestamps
a.getTime() < b.getTime(); // true

// ✅ También OK: operadores < >
a < b; // true (auto-conversión)
