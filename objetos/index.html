01
Creación de Objetos
Un objeto es una colección de pares clave-valor que agrupa datos y comportamientos relacionados.
Fundamental ▶ 
// Objeto literal: la forma más común y directa de crear objetos
const persona = {
  nombre: "Ana García",   // propiedad de tipo string
  edad: 28,              // propiedad de tipo number
  activo: true          // propiedad de tipo boolean
};

// Constructor new Object(): alternativa menos usada
const vehiculo = new Object();
vehiculo.marca = "Toyota";
vehiculo.año = 2023;

// Object.create(): crea un objeto con prototipo específico
const base = { tipo: "animal" };
const perro = Object.create(base);
perro.nombre = "Firulais";

// Mostrar resultados
console.log("Persona:", persona.nombre, "-", persona.edad, "años");
console.log("Vehículo:", vehiculo.marca, vehiculo.año);
console.log("Perro hereda tipo:", perro.tipo);

02
Acceso a Propiedades
Dos formas de leer o escribir propiedades: notación de punto (simple) y notación de corchetes (dinámica).
Fundamental ▶ 
const producto = {
  nombre: "Laptop",
  precio: 1299.99,
  "en-stock": true   // propiedad con guion requiere corchetes
};

// Notación de punto: rápida y legible para nombres simples
console.log(producto.nombre);   // → "Laptop"
console.log(producto.precio);   // → 1299.99

// Notación de corchetes: necesaria para nombres especiales o dinámicos
console.log(producto["en-stock"]);  // → true

// Acceso dinámico con variable: muy útil en funciones genéricas
const campo = "precio";
console.log(producto[campo]);   // → 1299.99 (accede dinámicamente)

// Propiedad inexistente devuelve undefined (sin error)
console.log(producto.color);    // → undefined

// Optional chaining (?.) evita errores con objetos nulos
const usuario = null;
console.log(usuario?.nombre ?? "Sin usuario"); // → "Sin usuario"

03
Métodos de Objeto
Los métodos son funciones almacenadas como propiedades de un objeto. Pueden usar this para acceder a sus propias propiedades.
Fundamental ▶ 
const calculadora = {
  historial: [],           // propiedad que almacena datos internos

  // Método con sintaxis abreviada (shorthand method)
  sumar(a, b) {
    const resultado = a + b;
    this.historial.push(`${a} + ${b} = ${resultado}`); // this → el objeto mismo
    return resultado;
  },

  // Arrow functions NO tienen su propio 'this'
  multiplicar: (a, b) => a * b,  // válida para cálculos sin 'this'

  // Método que accede a datos internos
  verHistorial() {
    return this.historial.join("\n");
  }
};

console.log(calculadora.sumar(5, 3));      // → 8
console.log(calculadora.sumar(10, 20));   // → 30
console.log(calculadora.multiplicar(4, 6)); // → 24
console.log(calculadora.verHistorial());

04
Destructuring y Spread
Destructuring extrae propiedades en variables. Spread (...) expande o combina objetos de forma concisa.
Intermedio ▶ 
const empleado = {
  id: 42,
  nombre: "Carlos",
  puesto: "Desarrollador",
  salario: 55000,
  departamento: "IT"
};

// Destructuring: extrae propiedades a variables locales
const { nombre, puesto, salario } = empleado;
console.log(`${nombre} → ${puesto}`);

// Renombrar al destructurar + valor por defecto
const { nombre: nombreCompleto, bonus = 0 } = empleado;
console.log(`Bonus de ${nombreCompleto}: $${bonus}`);

// Rest en destructuring: agrupa el resto de propiedades
const { id, ...datosPublicos } = empleado;
console.log("Sin ID:", datosPublicos);

// Spread: crea una copia superficial del objeto
const empleadoActualizado = { ...empleado, salario: 60000, nivel: "Senior" };
console.log("Salario nuevo:", empleadoActualizado.salario);

// Combinar dos objetos (el segundo sobreescribe propiedades repetidas)
const extra = { idiomas: ["JS", "Python"], remoto: true };
const perfil = { ...empleado, ...extra };
console.log("Perfil completo:", perfil.idiomas);

05
Métodos de Object
Object.keys(), Object.values() y Object.entries() permiten iterar y transformar objetos como si fueran arrays.
Intermedio ▶ 
const inventario = {
  manzanas: 50,
  naranjas: 30,
  uvas: 80,
  peras: 20
};

// Object.keys(): array con los nombres de las propiedades
const frutas = Object.keys(inventario);
console.log("Frutas:", frutas);

// Object.values(): array con los valores
const cantidades = Object.values(inventario);
const total = cantidades.reduce((acc, n) => acc + n, 0);
console.log("Total en stock:", total);

// Object.entries(): array de pares [clave, valor] — ideal para iterar
Object.entries(inventario).forEach(([fruta, cantidad]) => {
  const estado = cantidad > 40 ? "✓ OK" : "⚠ Bajo";
  console.log(`${fruta}: ${cantidad} unidades ${estado}`);
});

// Object.fromEntries(): convierte entries de vuelta a objeto
const doblado = Object.fromEntries(
  Object.entries(inventario).map(([k, v]) => [k, v * 2])
);
console.log("Stock duplicado:", doblado);

06
Constructores y Clases
Las clases son plantillas para crear múltiples objetos con la misma estructura. Son "azúcar sintáctica" sobre prototipos.
Intermedio ▶ 
// Clase base: define la estructura compartida
class Cuenta {
  // #saldo es una propiedad privada (solo accesible dentro de la clase)
  #saldo = 0;

  constructor(titular, saldoInicial) {
    this.titular = titular;          // propiedad pública
    this.#saldo = saldoInicial;       // inicializar privada
    this.movimientos = [];
  }

  depositar(monto) {
    this.#saldo += monto;
    this.movimientos.push(`+${monto}`);
  }

  retirar(monto) {
    if (monto > this.#saldo) throw new Error("Fondos insuficientes");
    this.#saldo -= monto;
    this.movimientos.push(`-${monto}`);
  }

  // Getter: propiedad de solo lectura que calcula un valor
  get saldo() { return this.#saldo; }

  toString() {
    return `[${this.titular}] Saldo: $${this.saldo}`;
  }
}

const cta = new Cuenta("María López", 1000);
cta.depositar(500);
cta.retirar(200);
console.log(cta.toString());
console.log("Movimientos:", cta.movimientos);

07
Herencia con Clases
Una subclase extiende otra con extends y llama al constructor padre con super(), heredando sus propiedades y métodos.
Avanzado ▶ 
// Clase padre: define comportamiento base
class Animal {
  constructor(nombre, especie) {
    this.nombre = nombre;
    this.especie = especie;
    this.vivo = true;
  }

  describir() {
    return `${this.nombre} es un ${this.especie}`;
  }

  hablar() { return "..."; } // método base a sobrescribir
}

// Subclase: extends hereda todo de Animal
class Perro extends Animal {
  constructor(nombre, raza) {
    super(nombre, "Canis lupus familiaris"); // llama al padre
    this.raza = raza;  // propiedad exclusiva de Perro
  }

  // Override: reemplaza el método del padre
  hablar() { return `${this.nombre} dice: ¡Guau!`; }

  fetch(objeto) { return `${this.nombre} trae el ${objeto} 🎾`; }
}

class Gato extends Animal {
  constructor(nombre) { super(nombre, "Felis catus"); }
  hablar() { return `${this.nombre} dice: Miau...`; }
}

const animales = [
  new Perro("Rex", "Labrador"),
  new Gato("Misi"),
  new Perro("Luna", "Beagle")
];

// Polimorfismo: cada animal ejecuta su propia versión de hablar()
animales.forEach(a => console.log(a.hablar()));
console.log(animales[0].fetch("palo"));
console.log("¿Rex es Animal?", animales[0] instanceof Animal);

08
Inmutabilidad y Object.freeze
Object.freeze() congela un objeto impidiendo cambios. Object.assign() copia propiedades de uno o más objetos fuente.
Avanzado ▶ 
// Objeto de configuración inmutable: evita modificaciones accidentales
const CONFIG = Object.freeze({
  API_URL: "https://api.ejemplo.com",
  TIMEOUT: 5000,
  VERSION: "2.1.0"
});

// Intentar modificar lanza error en strict mode, silencioso en normal
CONFIG.API_URL = "hackeado";  // ← no hace nada
console.log("URL (sin cambio):", CONFIG.API_URL);
console.log("¿Está congelado?", Object.isFrozen(CONFIG));

// Object.assign(): copia propiedades de objetos fuente al destino
const defaults = { color: "azul", tamaño: "M", cantidad: 1 };
const opciones = { color: "rojo", cantidad: 3 };  // sobreescribe

const pedido = Object.assign({}, defaults, opciones); // {} evita mutar defaults
console.log("Pedido final:", pedido);

// Patrón inmutable: actualizar sin mutar el original
const estadoAnterior = { usuario: "Ana", puntos: 100, nivel: 3 };
const nuevoEstado = { ...estadoAnterior, puntos: 150, nivel: 4 }; // spread para inmutabilidad
console.log("Estado anterior:", estadoAnterior.puntos);
console.log("Nuevo estado:", nuevoEstado.puntos);
